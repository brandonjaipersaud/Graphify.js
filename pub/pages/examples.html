Construct your own graph using the graph builder mode. The animation speed of Dijkstra's algorithm in this graph will be set to slow. 

  
        let config = {
            graphBuilder: true,
            styles: {
                        dimensions: [100, 400]
            }
        
        }
        
        let ctx = document.getElementsByClassName("container")[2]
        let graph = new Graph(config, ctx)
        graph.setDijkstraSpeed('slow')
        
        graph.visualize()

If you do not want to manually add vertices and edges to your graph, you can simply set the autoGenerate property to true and specify the number of vertices and edges that you want in the graph: 

  
    ctx = document.getElementsByClassName("container")[3]
    graph = new Graph({
        autoGenerate: true,
        styles: {
            dimensions: [100, 500]
        }
    }, ctx) 
            
    graph.visualize()
